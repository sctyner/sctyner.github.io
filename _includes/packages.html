R Packages

  geomnet - geomnet is a package built on top of the most recent major ggplot2 release. It provides a ggplot2 geom called geom_net to visualize graphs and networks. It also include the function stat_net to calculate network layouts with the sna package. Finally, the function geom_circle is included to draw circles using ggplot2.

The vignettes provide code necessary to reproduce examples, speed tests, and figures in the paper. The package contains a single function timeDrawings() for users to compare the drawing time of many different network visualization implementations, as well as allowing the user to input their own code to compare to the others.
